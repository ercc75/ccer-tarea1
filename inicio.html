¡Bienvenido a la Unidad 1 de Desarrollo web en entorno servidor!


En esta web se expondrán 5 ejercicios correspondientes a la Unidad 1 de Desarrollo web en entorno servidor. Podremos acceder a cada uno de ellos con sus correspondientes apartados a través de la barra superior de navegación.

En el ejercicio 1 veremos una serie de conceptos previos, tales como:

  • Diferencias entre desarrollador front-end, back-end y full-stack
  • Qué es un framework CSS, qué es Boostrap y qué otros frameworks CSS hay
  • Qué es WCAG y la importancia de la validación como valor añadido
  • Qué es un servidor web, qué es el protocolo HTTP, qué es Apache y qué es XAMPP

En el ejercicio 2 veremos un tutorial de instalación de XAMPP en Windows y Linux, indicando además cómo se ejecuta código PHP embebido y código PHP puro


En el ejercicio 3 veremos dos tipos de formularios:

  • Formulario de recogida y validación de datos de usuario, para su posterior presentación por pantalla
  • Formulario de recogida y validación de datos de usuario para la generación de un CV, para su posterior presentación por pantalla

En el ejercicio 4 probaremos las siguientes aplicaciones web:

  • Juego de piedra, papel, tijera, lagarto o Spock
  • Cálculo del área y longitud de una circunferencia, una vez introducido el radio
  • Informe de salud personal, en el que se mostrará el IMC y el metabolismo basal, previa introducción de unos determinados datos

En el ejercicio 5 veremos algunos conceptos sobre el sistema de control de versiones:

  • Qué es un sistema de control de versiones y qué tipos de sistemas son los más utilizados
  • Qué es Git, qué es Github y en qué se diferencian
  • Cómo se hace una cuenta en Github
  • Uso básico de Github: cómo se sube un proyecto
